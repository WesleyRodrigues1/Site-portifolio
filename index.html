MINHAS HABILIDADE.

HTML

HTML (Hypertext Markup Language) é a linguagem padrão para a criação e estruturação de conteúdo na web. Ele é usado para definir a estrutura de uma página da web, como cabeçalhos, parágrafos, links e imagens. Com o HTML, você pode criar documentos web interativos e bem organizados

CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a aparência e o layout de elementos HTML em uma página da web. Com o CSS, você pode definir cores, fontes, tamanhos, margens e posições de elementos para criar um design visualmente atraente e consistente

JAVASCRIPT

JavaScript é uma linguagem de programação amplamente usada na web para tornar as páginas interativas e dinâmicas. Com o JavaScript, você pode criar funcionalidades como formulários interativos, animações, validação de entrada e muito mais. Ele é executado no navegador do cliente e permite que os sites respondam às ações do usuário em tempo real.

PYTHON

Python é uma linguagem de programação versátil e fácil de aprender, usada em desenvolvimento de software, análise de dados, automação e muitas outras aplicações. Com Python, é possível criar scripts, aplicativos desktop e web, além de trabalhar com uma ampla variedade de bibliotecas para facilitar o desenvolvimento de soluções de software.

MYSQL

MySQL é um sistema de gerenciamento de banco de dados relacional (RDBMS) de código aberto amplamente utilizado. Ele é usado para armazenar, gerenciar e recuperar dados de maneira eficiente em aplicativos e sites. O MySQL utiliza a linguagem SQL (Structured Query Language) para consultar e manipular dados. Ele oferece recursos de segurança, escalabilidade e desempenho, tornando-o uma escolha popular para aplicativos que requerem armazenamento de dados confiável.

ANGULAR

Angular é um framework de desenvolvimento front-end de código aberto desenvolvido pelo Google. Ele é usado para criar aplicativos web dinâmicos e interativos. O Angular utiliza a linguagem TypeScript, que é uma superset do JavaScript, e segue o modelo de design de arquitetura Modelo-Visão-Controlador (MVC)..
